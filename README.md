## Clone this project

```bash
> git clone https://github.com/TxC-Toxic/SatanV8
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd SatanV8
> bash install.sh
```

### Usage
```bash
> npm start
```
